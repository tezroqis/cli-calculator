# CLI Калькулятор

Простой калькулятор для командной строки, написанный на Python. Поддерживает базовые арифметические операции.

## Возможности

- Сложение (`+`)
- Вычитание (`-`)
- Умножение (`*`)
- Деление (`/`) с защитой от деления на ноль

## Запуск

```bash
python calc.py
```

## Использование

```
=== CLI Калькулятор ===

Доступные операции: +  -  *  /
Введите операцию (или 'q' для выхода): +
Введите первое число: 10
Введите второе число: 5
Результат: 15.0
```

Для выхода из программы введите `q`.

## Требования

- Python 3.6 или выше
