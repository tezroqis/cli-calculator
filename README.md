# CLI Калькулятор v2.0

Продвинутый калькулятор для командной строки на Python. Поддерживает разбор строковых выражений, математические функции, константы и интерактивный REPL-режим.

## Возможности

### Базовые операции
- Сложение (`+`), вычитание (`-`), умножение (`*`), деление (`/`)
- Возведение в степень (`^` или `**`)
- Остаток от деления (`mod`)
- Проценты (`%`)

### Функции
- `sqrt(x)` — квадратный корень
- `abs(x)` — модуль числа
- `sin(x)`, `cos(x)`, `tan(x)` — тригонометрические функции (радианы)
- `log(x)`, `ln(x)` — логарифмы (десятичный и натуральный)
- `round(x)` — округление до целого
- `factorial(x)` или `x!` — факториал

### Константы
- `pi` — число π (3.14159...)
- `e` — число Эйлера (2.71828...)

### Прочее
- Скобки для группировки: `( )`
- Унарный минус: `-5`, `3 + (-2)`
- Приоритет операторов согласно математическим правилам

## Установка

Требуется **Python 3.10+**.

```bash

```

### Из репозитория

```bash
git clone https://github.com/tezroqis/cli-calculator.git
cd cli-calculator
pip install -e .
```

### Зависимости

- `colorama` (опционально, для цветного вывода)

## Использование

### Интерактивный режим (REPL)

```bash
calc
# или
python calc.py
```

В REPL-режиме можно вводить выражения и сразу получать результат:

```
>>> 2 + 3 * 4
  = 14
>>> sqrt(16) + sin(pi / 2)
  = 5
>>> 5!
  = 120
>>> 2 ^ 3 ^ 2
  = 512
```

Специальные команды REPL:

| Команда       | Действие              |
|---------------|-----------------------|
| `:h`, `:help` | Показать справку      |
| `:hist`       | История вычислений    |
| `:q`, `:quit` | Выход                 |
| `Ctrl+C`      | Выход                 |

### Вычисление одного выражения

```bash
calc "2 + 3 * 4"
calc -e "sqrt(25) + log(100)"
python calc.py "(5 + 3)! / 2"
```

### Аргументы командной строки

```
calc [-h] [-e ВЫРАЖЕНИЕ] [-i] [--no-color] [выражение]

Позиционные аргументы:
  выражение          Математическое выражение

Опции:
  -e, --expression   Выражение для вычисления
  -i, --interactive  Запустить REPL
  --no-color         Отключить цветной вывод
  -h, --help         Показать справку
```

## Примеры выражений

| Выражение                     | Результат |
|-------------------------------|-----------|
| `2 + 3 * 4`                   | 14        |
| `(2 + 3) * 4`                 | 20        |
| `2 ^ 3 ^ 2`                   | 512       |
| `sqrt(16) + sin(pi / 2)`      | 5.0       |
| `10 mod 3`                    | 1         |
| `10 % 200`                    | 20        |
| `5!`                          | 120       |
| `factorial(5)`                | 120       |
| `abs(-3.14)`                  | 3.14      |
| `round(3.7)`                  | 4         |
| `ln(e)`                       | 1.0       |
| `log(100)`                    | 2.0       |
| `3 + 4 * 2 / (1 - 5) ^ 2`    | 3.5       |
